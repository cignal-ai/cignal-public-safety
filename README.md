# Cignal DICOM Viewer - Bug Reporting

This repository is dedicated to collecting bug reports and issues for the Cignal DICOM Viewer.

**Please DO NOT submit pull requests to this repository.** This repository is for issue tracking only.

## How to Report a Bug

If you encounter a bug or an unexpected behavior in the Cignal DICOM Viewer, please open a new issue here. When reporting a bug, please include:

1.  A clear and concise description of the bug.
2.  Steps to reproduce the behavior.
3.  Expected behavior.
4.  Screenshots or videos (if applicable).
5.  Your operating system and browser information.

## Main Project

For information about the Cignal DICOM Viewer project itself, including documentation, features, and other details, please refer to the [main project repository](https://github.com/KitwareMedical/VolView) (or a relevant link if different).
